## ⚡ Quick Pattern Recognition Guide

```
Input is SORTED ARRAY + find pair/triplet?         → TWO POINTERS
Subarray/substring with constraint?                → SLIDING WINDOW
Find position in sorted / minimize-maximize?       → BINARY SEARCH
Linked list + cycle/middle/reverse?                → SLOW-FAST POINTER
Parentheses/next greater/min in stack?             → MONOTONIC STACK
Tree + height/path/traversal?                      → TREE DFS/BFS
Shortest path in graph?                            → BFS (unweighted) / DIJKSTRA (weighted)
Top K / Kth largest?                               → HEAP
All combinations/permutations/subsets?             → BACKTRACKING
Count ways / optimal cost with choices?            → DYNAMIC PROGRAMMING
Intervals / scheduling / locally optimal?          → GREEDY
Prefix search / word search?                       → TRIE
Connected components / redundant edges?            → UNION-FIND
Range sum/min/max queries with updates?            → SEGMENT TREE / BIT
XOR / single number / set/clear bits?             → BIT MANIPULATION
```
